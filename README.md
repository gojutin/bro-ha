<h2 id="demo">Run locally</h2>

```sh
$ git clone git@github.com:gojutin/styled-logs.git
$ cd styled-logs/demo
$ yarn install (or npm install)
$ yarn start (or npm start)
```
Open your browser and visit `http://localhost:3000`
